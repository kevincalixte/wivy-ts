
# WIVY

### Contexte et objectifs
Application web qui permet de mettre en relation des adultes, les profils utilisateurs s'affichent selon la localisation

#### Utilisateurs finaux
Les utilisateurs finaux sont :
- Des adultes de plus de 18 ans
Les utilisateurs peuvent :
- S'inscrire et se connecter sur leur profil

#### Administrateurs
##### Modérateurs
Le modérateur peut :
- Verifier la photo de profil
- Bloquer un compte qui ne respecte pas les règles d'utilisation
- Débloquer un compte

##### Administrateur
L'administrateur peut :
- Créer, modifier ou supprimer un compte
- Voir l'activité du site (trafic, statistiques)

## Page principale
- Accueil : Offres d'abonnement, avis clients
- Inscription/Connexion : Formulaire, authentification Supabase
- offre d'abonnement
- Avis clients : Affichage des avis depuis la base de donnée
- Contact : Formulaire de contact, lien réseaux sociaux

## Page utilisateur connecté
- Profils utilisateurs : Affichage des informations (identité, image, description, statut)
- Navigation : Barre haut et bas
- Recherche par tags
- Liker un profil
- Mettre un profil en favoris
- Messagerie : Chat privé
- Notification : visite profil/likes/favoris/message

## Contraintes 
- Next.js, TypeScript, Supabase, TailwindCSS
- Responsive
- Temps

## Sécurité
- Hash de mots de passe
- Requêtes préparées
